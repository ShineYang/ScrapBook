## Features

#### Clipboard

Save your clipboard history, offer text editor and pullword.

#### Easy to manage

Manage cliped text by app name.

#### Share text

 Share clips to apps or creat QRCode.

#### Notification widget

 Provide notification widget.

#### Design

 Full material design.
